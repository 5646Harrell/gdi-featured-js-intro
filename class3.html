While loops

while will repeat the same code over and over until some condition is met.

var bottlesOfBeer = 99;

while (bottlesOfBeer > 0) {
  console.log(bottlesOfBeer + ' bottles of beer on the wall');
  bottlesOfBeer = bottlesOfBeer - 1;
}

Infinite Loops

Make sure something changes in the loop, or your loop will go on forever...

Photo credit: Samuel John cc

For loops

for loops are very similar, but you declare a counter in the statement.

// will count 1 to 10
for (var i = 1; i <= 10; i++) {
  console.log(i);
}

Loops and logic

You can add other statements or logical operators inside the loops.

// Count from 1 to 50

for (var i = 1; i <= 50; i++) {
  console.log(i);

  // Says 'Fizz' after multiples of three
  if (i % 3 == 0) {
    console.log(' Fizz');
  }

  // Says 'Buzz' after multiples of five
  if (i % 5 == 0) {
    console.log(' Buzz');
  }
}

Break

To exit a loop, use the break statement.

// Count from 100 to 200
for (var i = 100; i <= 200; i++) {
  console.log('Testing ' + i);

  //Stop at the first multiple of 7
  if (i % 7 == 0) {
    console.log('Found it! ' + i);
    break;
  }
}

Arrays

Photo credit: Jesus Solana cc

Arrays

Arrays are ordered lists of values.

var arrayName = [element0, element1, ...];

You can put different types of data into an array.

var rainbow = ['Red', 'Orange', 'Yellow', 'Green',
  'Blue', 'Indigo', 'Violet'];

var lotteryNumbers = [33, 72, 64, 18, 17, 85];

var myFavoriteThings = ['Pizza', 33, ['Ponyo', 'Spirited Away']];

Array Length

The length property tells you how many things are in an array

var directors = ['Coppola', 'Miyazaki', 'Kubrick'];

console.log(directors.length);

Using Arrays

You can access items with "bracket notation" by using the position of the item you want.

var directors = ['Coppola', 'Miyazaki', 'Kubrick'];

var firstDirector = directors[0];
var lastDirector  = directors[2];

JS arrays are "zero-indexed", so starts counting at 0.

Changing arrays

You can use bracket notation to change an item in an array

var parrots = ['Cockatiel', 'Amazon', 'Eclectus'];

parrots[0] = 'African Grey';

Expanding arrays

Arrays do not have a fixed length. You can use push to add something to an array.

var parrots = ['Cockatiel', 'Amazon', 'Eclectus'];

parrots.push('Cockatoo');

Iterating through arrays

Use a for loop to easily work with each item in an array.

var favoriteFoods = ['pizza', 'tacos', 'lasagna'];

for (var i = 0; i < favoriteFoods.length; i++) {
  console.log(favoriteFoods[i]);
}

Objects

Photo credit: Jirí Volejník cc

Objects

Objects let us store a collection of properties.

var objectName = {
  propertyName: propertyValue,
  propertyName: propertyValue,
  ...
};

var aboutMe = {
  hometown: 'Columbus, GA',
  eyeColor: 'green',
  pets: ['cat', 'parrot'],
  birthday: {month: 11, day: 17}
};

Accessing Objects

You can retrieve values using "dot notation"

var aboutMe = {
  hometown: 'Columbus, GA',
  eyeColor: 'green'
};

var myHometown = aboutMe.hometown;

Or using "bracket notation" (like arrays)

var myEyes = aboutMe['eyeColor'];

Changing Objects

You can use dot or bracket notation to change properties

var aboutMe = {
  hometown: 'Columbus, GA',
  eyeColor: 'Auburn'
};

aboutMe.eyeColor = 'blue';

Add new properties

aboutMe.married = true;

Or delete them

delete aboutMe.married;

aboutMe.married = undefined;

Arrays of Objects

Because arrays can hold any data type, they can also hold objects

var myPets = [
  {name: 'Albert', age: 21},
  {name: 'Alex',   age: 6}
];

for (var i = 0; i < myPets.length; i++) {
  var pet = myPets[i];
  console.log(pet.name + ' is ' + pet.age + ' years old.');
}

Objects

Just like other data types, objects can be passed into functions:

var albertTheParrot = {
  age: 21,
  color: 'white',
  likes: ['ravioli', 'screwdrivers'],
  birthday: {month: 3, day: 14, year: 1995}
}

function describeParrot(parrot) {
  console.log('This parrot is ' + parrot.age + ' years old with '
  + parrot.color + ' feathers.');
}

describeParrot(albertTheParrot);

Object methods

Objects can also hold functions.

var albertTheParrot = {
  age: 21,
  color: 'white',
  talk: function() {
    console.log('Hello!');
  },
  eat: function(food) {
    console.log('Yum, I love ' + food);
  }
};

Call object methods using dot notation:

albertTheParrot.talk();

albertTheParrot.eat('ravioli');
